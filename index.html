Visão Geral do Sistema de IA
Descrição

A IA do Grupo Monte é um agente conversacional baseado em LLM (Large Language Model) implantado no canal WhatsApp, integrado ao ERP SHX e ao CRM Bitrix24. Atua como primeiro ponto de contato digital para as 9 lojas das bandeiras Monte e Tropical Center.

2
Fluxo de Conversação — Lógica Principal
Classificação de Intenção (obrigatório)

A cada mensagem recebida, a IA deve classificar a intenção antes de responder:

IntençãoExemploFluxo acionado
Consulta de produto"Tem porcelanato 60x60?"API Catálogo SHX → produto + preço
Consulta financeira"Quero segunda via do boleto"API Títulos SHX → consulta CPF/CNPJ
Localização/unidade"Qual loja mais perto do Zerão?"Roteamento → unidade + horário
Rastreamento de pedido"Meu pedido chegou?"API Logística → status (N4)
Saudação / indefinido"Oi", "Bom dia"Saudação padrão + menu de opções
Escalada humana"Quero falar com um vendedor"Transfer para Bitrix24 → notifica atendente
Regra de Escalada (obrigatório)

A IA deve escalar para atendente humano: (1) cliente solicitar explicitamente, (2) após 3 tentativas sem resolução, (3) intenção de compra com valor acima do threshold configurável, (4) reclamação ou insatisfação identificada no sentimento.

Roteamento por Unidade
if (cliente.regiao in ["Tiradentes","Norte","Congós"]) → Monte Tiradentes if (cliente.regiao in ["Macapá Centro","Perpétuo"]) → Monte Matriz if (cliente.regiao in ["Laguinho","Santa Rita"]) → Tropical Laguinho if (cliente.regiao in ["Beirol","Buritizal"]) → Tropical Beirol if (cliente.regiao in ["Santana","Elesbão"]) → Monte Santana // fallback: perguntar CEP → calcular unidade mais próxima

Requisitos de chunking e embedding

Produtos: cada SKU = 1 chunk com campos normalizados. Chunking semântico para FAQs (1 pergunta + resposta = 1 chunk). Meta: ≥95% de cobertura das 5 principais famílias em testes.

5
Integração Bitrix24 — Leads e CRM
Critérios para criação automática de lead

Lead criado quando: (1) cliente informa nome + produto de interesse, (2) intenção de compra com confiança ≥ 0.75, (3) atendimento qualificado concluído. Atendimentos financeiros também geram registro na fila correta.

Campos obrigatórios no lead
Campo Bitrix24FonteObrigatório
TITLE"[IA] {Nome} — {Produto}"MUST
NAME / LAST_NAMEExtraído da conversa pelo LLMMUST
PHONENúmero WhatsApp do clienteMUST
ASSIGNED_BY_IDVendedor da unidade roteadaMUST
COMMENTSHistórico resumido da conversa (LLM)MUST
UF_CRM_PRODUTOSKU consultado na sessãoSHOULD
UF_CRM_UNIDADEUnidade roteada pelo sistemaSHOULD
